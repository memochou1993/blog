## 做法

建立專案。

```BASH
mkdir go-vue-template
cd go-vue-template
```

初始化 Go Modules。

```BASH
go mod init github.com/memochou1993/go-vue-template
```

建立前端專案。

```BASH
vue create app
```

編譯前端專案。

```BASH
yarn run build
```

新增 `main.go` 檔。

```GO
package main

import (
	"embed"
	"io/fs"
	"log"
	"net/http"
)

//go:embed ui/dist
var ui embed.FS

func main() {
	stripped, err := fs.Sub(ui, "ui/dist")
	if err != nil {
		log.Panic(err)
	}
	http.Handle("/", http.FileServer(http.FS(stripped)))

	log.Panic(http.ListenAndServe(":8000", nil))
}
```

編譯執行檔。

```BASH
go build
```
