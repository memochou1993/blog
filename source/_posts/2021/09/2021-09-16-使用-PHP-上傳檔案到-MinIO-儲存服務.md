## 安裝套件

安裝 `aws/aws-sdk-php` 套件。

```BASH
composer require aws/aws-sdk-php
```

## 存取檔案

建立一個 `S3Client` 實體。

```PHP
$s3 = new \Aws\S3\S3Client([
    'version' => 'latest',
    'region'  => 'us-east-1',
    'endpoint' => 'http://127.0.0.1:9000',
    'use_path_style_endpoint' => true,
    'credentials' => [
        'key'    => '', // KEY
        'secret' => '', // SECRET
    ],
]);
```

上傳檔案。

```PHP
$insert = $s3->putObject([
    'Bucket' => 'bucket',
    'Key'    => 'test.txt',
    'Body'   => 'Hello from Minio',
]);
```

下載檔案。

```PHP
$retrieve = $s3->getObject([
    'Bucket' => 'bucket',
    'Key'    => 'test.txt',
    'SaveAs' => 'test.txt',
]);

echo $retrieve['Body'];
```
