## 安裝

使用以下指令下載 `air` 執行檔。

```BASH
curl -sSfL https://raw.githubusercontent.com/cosmtrek/air/master/install.sh | sh -s -- -b $(go env GOPATH)/bin
```

下載後，將執行檔加入至使用者的執行檔目錄中。

## 使用

建立專案。

```BASH
mkdir test-project
cd test-project
```

啟用 Go Modules。

```BASH
go mod init
```

使用 `air` 指令建立一個 `.air.toml` 檔。

```BASH
air init
```

啟用熱重載。

```BASH
air
```
