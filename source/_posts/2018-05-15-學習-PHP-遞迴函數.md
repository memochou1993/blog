## 等差數列的合
```PHP
function total($num) {
    if ($num > 1) {
        $value = $num + total($num - 1);
    } else {
        return 1;
    }

    return $value;
}
```

## 正整數的階乘
```PHP
function total($num) {
    if ($num > 1) {
        $value = $num * total($num - 1);
    } else {
        return 1;
    }

    return $value;
}
```

## 平方的總和
```PHP
function total($num) {
    if ($num > 1) {
        $value = $num * $num + total($num - 1);
    } else {
        return 1;
    }

    return $value;
}
```
