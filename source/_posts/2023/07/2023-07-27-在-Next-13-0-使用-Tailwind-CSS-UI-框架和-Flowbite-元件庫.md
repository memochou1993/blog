## 前言

本文要安裝的 [Flowbite](https://flowbite.com/) 是基於 Tailwind CSS UI 框架的元件庫。另外還有 [Flowbite React](https://www.flowbite-react.com/) 元件庫可以使用，但是元件較少。

## 建立專案

建立專案，同時安裝 Tailwind CSS 套件。

```bash
npx create-next-app@latest

✔ What is your project named? … example
✔ Would you like to use TypeScript? … No
✔ Would you like to use ESLint? … Yes
✔ Would you like to use Tailwind CSS? … Yes
✔ Would you like to use `src/` directory? … No
✔ Would you like to use App Router? (recommended) … Yes
✔ Would you like to customize the default import alias? … No
```

安裝 Flowbite 元件庫。

```bash
npm install flowbite
```

修改 `tailwind.config.js` 檔。

```js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    // ...
    'node_modules/flowbite-react/**/*.{js,jsx,ts,tsx}', // 添加此行
  ],
  // ...
  plugins: [
    require('flowbite/plugin'), // 添加此行
  ],
};
```

修改 `app/layout.js` 檔，引入 Flowbite 的互動腳本。

```js
import './globals.css';
import { Inter } from 'next/font/google';
import Script from 'next/script';

const inter = Inter({ subsets: ['latin'] });

export const metadata = {
  title: 'Create Next App',
  description: 'Generated by create next app',
};

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body className={inter.className}>
        {children}
        <Script src="https://cdnjs.cloudflare.com/ajax/libs/flowbite/1.7.0/flowbite.min.js" />
      </body>
    </html>
  );
}
```

修改 `app/layout.js` 檔，在裡面使用 `Header` 元件。

```js
import Header from '@/components/header';
import './globals.css';
import { Inter } from 'next/font/google';
import Script from 'next/script';

const inter = Inter({ subsets: ['latin'] });

export const metadata = {
  title: 'Create Next App',
  description: 'Generated by create next app',
};

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body className={inter.className}>
        <Header />
        {children}
        <Script src="https://cdnjs.cloudflare.com/ajax/libs/flowbite/1.7.0/flowbite.min.js" />
      </body>
    </html>
  );
}
```

啟動服務。

```bash
npm run dev
```
