## 簡介

nmap 是一個命令列工具，用來掃描主機是否有開機，並且開啟哪些連接埠號。

## 使用

掃描單一主機。

```bash
nmap 192.168.1.1
```

掃描多個主機。

```bash
nmap 192.168.1.1 192.168.1.2 192.168.1.3
```

執行快速掃瞄（僅掃描常用埠號）。

```bash
nmap -F 192.168.1.1
```
