## 做法
新增 `ec2.sh` 檔：
```BASH
#!/bin/bash

# 金鑰位置
key="~/.ssh/aws.pem"

# 主機選項
user[1]="ubuntu"
user[2]="deployer"

# 使用者選項
  ip[1]="xx.xxx.xxx.xxx"
  ip[2]="yy.yyy.yyy.yyy"

for ((i=1; i <= ${#user[@]}; i++))
do
    echo "${i})" ${user[$i]}
done

# 讀取主機選項
read input[1]

user=${user[${input[1]}]}

for ((i=1; i <= ${#ip[@]}; i++))
do
    echo "${i})" ${ip[$i]}
done

# 讀取使用者選項
read input[2]

ip=${ip[${input[2]}]}

echo "Connecting to ${user}@${ip}...\n"

# 執行連線
ssh -i "~/.ssh/aws.pem" "${user}@${ip}"

exit 0
```

進行連線。
```
$ sh ec2.sh
```
