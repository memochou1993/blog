## 安裝

使用 `brew` 安裝 Go。

```BASH
brew install go
```

修改 `~/.zshrc` 檔：

```BASH
export GO111MODULE=on
export GOPATH=$HOME/Workspace/go
export GOBIN=$HOME/Applications/go/bin
export GOMODCACHE=$HOME/Applications/go/pkg/mod
export PATH=$GOBIN:$PATH
```

重新加載啟動文件。

```BASH
source ~/.bashrc
```

確認 Go 的版本：

```BASH
go version
```

查看 Go 的環境變數：

```BASH
go env
```

## 升級

使用 `brew` 指令升級 Go。

```BASH
brew upgrade go
```
