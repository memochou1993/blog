## 作法
修改 `package.json` 檔：
```JSON
{
    ...
    "scripts": {
        "build": "hexo deploy --generate && git add . && git commit -m"
    },
    ...
}
```

## 發布與推送
```
$ npm run build -- "新增文章"
> hexo deploy --generate && git add . && git commit -m "新增文章"
$ git push
```
