## 遠端
查看倉庫。
```
$ git remote -v
origin  git@github.com:memochou1993/json.git (fetch)
origin  git@github.com:memochou1993/json.git (push)
```

修改倉庫地址。
```
$ git remote set-url origin git@github.com:memochou1993/json-vue.git
$ git remote -v
origin  git@github.com:memochou1993/json-vue.git (fetch)
origin  git@github.com:memochou1993/json-vue.git (push)
```
