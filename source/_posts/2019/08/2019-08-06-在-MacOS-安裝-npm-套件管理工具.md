## 步驟

使用 `curl` 指令安裝 nvm 版本切換工具。

```CMD
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
```

確認 nvm 的版本。

```CMD
nvm --version
```

重新讀取 `.bashrc` 檔。

```CMD
source .bashrc
```

使用 nvm 指令安裝 npm 的穩定版本。

```CMD
nvm install --lts
```

使用 nvm 指令指定 npm 的穩定版本。

```CMD
nvm use --lts
```

確認 npm 的版本。

```CMD
npm --version
```
