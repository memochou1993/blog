## 設定

設定使用者名稱。

```CMD
git config --global user.name <USERNAME>
```

設定電子郵件地址。

```CMD
git config --global user.email <EMAIL>
```

## 遠端

查看倉庫。

```CMD
git remote -v
```

修改倉庫地址。

```CMD
git remote set-url origin <URL>
```
