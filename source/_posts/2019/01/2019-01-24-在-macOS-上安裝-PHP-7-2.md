## 安裝

使用 `brew` 指令安裝 PHP。

```BASH
brew install php@7.2
```

啟動 PHP 服務。

```BASH
brew services start php@7.2
```

關閉 PHP 服務。

```BASH
brew services stop php@7.2
```
