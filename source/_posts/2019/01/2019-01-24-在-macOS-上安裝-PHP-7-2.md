## 安裝

使用 `brew` 安裝 PHP 7.2。

```bash
brew install php@7.2
```

啟動 PHP-FPM 服務。

```bash
brew services start php@7.2
```

關閉 PHP-FPM 服務。

```bash
brew services stop php@7.2
```

## 環境變數

修改 `.zshrc` 檔：

```bash
export PATH=/usr/local/opt/php@7.2/bin:$PATH
```

重新讀取環境變數。

```bash
exec $SHELL
```
