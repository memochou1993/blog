## 設定

設定使用者名稱。

```bash
git config --global user.name <USERNAME>
```

設定電子郵件地址。

```bash
git config --global user.email <EMAIL>
```

## 遠端

查看遠端儲存庫。

```bash
git remote -v
```

新增遠端儲存庫。

```bash
git remote add origin <URL>
```

修改遠端儲存庫。

```bash
git remote set-url origin <URL>
```
