## 堆疊

堆疊是一種 LIFO (Last In, First Out) 的資料結構。

```py
>>> stack = []
>>> stack.append(3)
    stack.append(45)
    stack.append(27)
    stack.append(91)
    stack
[3, 45, 27, 91]
>>> len(stack)
4
>>> stack[-1]
91
>>> stack.pop()
91
>>> len(stack) != 0
True
```
