## 做法

建立專案。

```BASH
npx create-vuepress-site my-project
```

進到 `docs` 資料夾。

```BASH
cd my-project/docs
```

安裝依賴套件。

```BASH
npm install
```

啟動服務。

```BASH
npm run dev
```

前往：<http://localhost:8080/>
