## 前言

GitHub 使用 [linguist](https://github.com/github/linguist) 來偵測專案使用的程式語言，透過設定，可以修改專案中程式語言的使用比例。

## 做法

在專案的根目錄新增 `.gitattributes` 檔。

```bash
touch .gitattributes
```

修改 `.gitattributes` 檔，排除指定資料夾底下的所有檔案。

```env
vendor/** linguist-vendored
```
