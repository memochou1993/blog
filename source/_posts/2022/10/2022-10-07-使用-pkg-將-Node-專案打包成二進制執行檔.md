## 做法

安裝依賴套件。

```bash
npm install pkg -D
```

修改 `package.json` 檔。

```json
{
  // ...
  "scripts": {
    "build": "pkg index.js -o my-app",
  },
  // ...
}
```

執行編譯。

```bash
npm run build
```

執行二進制執行檔。

```bash
./my-app
```
