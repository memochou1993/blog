## 建立專案

建立專案。

```BASH
mkdir storybook-react
cd storybook-react
```

初始化專案。

```BASH
npm init
```

## 安裝依賴套件

安裝 Vite 工具。

```BASH
npm i vite -D
```

安裝 React 框架。

```BASH
npm i react@17.0.0 react-dom@17.0.0 -D
```

安裝 Storybook 工具。

```BASH
npx sb@latest init
```

## 啟動介面

啟動 Storybook 介面。

```BASH
npm run storybook
```
