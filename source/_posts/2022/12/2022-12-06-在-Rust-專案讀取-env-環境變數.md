## 做法

安裝依賴套件。

```toml
[dependencies]
dotenvy = { version = "0.15" }
```

修改 `main.rs` 檔。

```rs
use dotenvy::dotenv;
use std::env;

fn main() {
    dotenv().expect(".env file not found");

    for (key, value) in env::vars() {
        println!("{key}: {value}");
    }
}
```
