## 做法

新增 `main.go` 檔：

```GO
package main

import (
	"log"
	"net/http"
)

func main() {
	http.Handle("/", http.FileServer(http.Dir("./web/dist/")))

	log.Fatal(http.ListenAndServe(":8080", nil))
}
```

建立 Vue 專案。

```BASH
vue create web
```

執行編譯。

```BASH
cd web && yarn build
```

啟動應用程式。

```BASH
go run main.go
```

前往：<http://localhost:8080>
