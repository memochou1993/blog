## 做法

重複地走訪要排序的數列，相鄰比較兩個元素，再根據條件進行交換。

## 實作

```GO
func BubbleSort(items []int) []int {
	// 疊代每個元素
	for i := 0; i < len(items)-1; i++ {
		// 疊代未排序過的每個元素
		for j := 0; j < len(items)-1-i; j++ {
			// 判斷相鄰的兩個元素
			if items[j] > items[j+1] {
				// 交換兩個元素
				temp := items[j]
				items[j] = items[j+1]
				items[j+1] = temp
			}
		}
	}

	return items
}
```
