## 做法

把未排序的元素，在已排序的陣列裡頭，從後向前比對，找到相應的位置後插入。

## 實作

```GO
func InsertionSort(items []int) []int {
	// 從第二個元素開始疊代
	for i := 1; i < len(items); i++ {
		// 記錄當前元素
		temp := items[i]

		// 從後向前比對
        j := i - 1
		for ; j >= 0 && temp < items[j]; j-- {
			// 將已排序的陣列中的元素向後複製
			items[j+1] = items[j]
		}

		// 將當前元素插入
		items[j+1] = temp
	}

	return items
}
```
